# IntellMeet

A full-stack AI-powered video meeting platform.

## Features

- JWT Authentication
- Meeting Creation & Management
- Video Calling (WebRTC)
- Real-time Chat (Socket.io)
- Meeting Recording
- Screen Sharing
- AI Meeting Summary
- Action Item Extraction
- Meeting History

## Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Node.js
- Express
- MongoDB
- Socket.io
- Redis
